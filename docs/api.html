Estimate the ground plane from a point cloud.

The plane is not necessarily the ground. It is just a plane whose normal is roughly parallel to some up vector.

Parameters

{np.ndarray} -- Input point cloud (pts) –

Keyword Arguments

  • {float} -- The threshold for the RANSAC (default (residual_threshold) – {0.5})

  • {int} -- Number of ransac iterations (default (ransac_iter) – {100})

  • {float} -- The normal threshold to determine potential (normal_threshold) – ground plane points (default: {0.85})

  • {np.ndarray} -- The up vector indicating an approximate normal vector (up) – for the ground plane.

  • {np.ndarray} -- Point validity mask (only the points (mask) – where mask == True are used (default: None))

Returns

[np.ndarray] – Array of the indices of the points that belong to the

ground plane

api.utils.estimate_plane(pts, residual_threshold=0.5, ransac_iter=1000, min_samples=3)¶

Estimate a plane from a 3D points array.

Plane parameters are estimated using this equation

a*x + b*y + c*z + d = 0

returned parameters are in the form

[a, b, c, d]

Parameters

{np.ndarray} -- The points array (pts) –

Keyword Arguments

  • {float} -- The ransac residual threshold (default (residual_threshold) – {0.5})

  • {int} -- Number of ransac iterations (default (ransac_iter) – {100})

Returns

(np.ndarray, np.ndarray, np.ndarray) – Plane parameters, inlier mask, signed distance to plane
